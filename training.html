Training Guide

Using the Repair Description & Fixed Comment Tools

Why Standardization Matters

Using standardized descriptions and comments is one of the most important things we can do to improve our efficiency and service quality. When everyone uses the same format, we create clean, reliable data.

  • Faster Analysis: Clean data allows us to quickly identify trends, like a specific part failing more often than others.
  • Better Inventory Management: We can accurately track which parts are used most frequently and ensure we always have them in stock.
  • Improved Consistency: Every customer receives clear and consistent communication about their repair.
  • Saves Time: These tools make logging your work faster and easier, eliminating guesswork.

How to Use the Repair Description Generator

This tool is for creating the main repair description when you have replaced one or more physical parts.

1

Select Your Language

Choose EN, FR, or NL at the top. The tool will translate everything for you.

2

Choose a Category

Use the tabs (Regular Device, Foldable, etc.) to find the components you worked on.

3

Select All Replaced Parts

Click on every part you replaced. The tool will remember your selections even if you switch tabs (e.g., selecting a Display from "Regular Device" and a Screen Protector from "Services").

4

Copy the Description

The description is generated automatically in the text box at the bottom. Click the "Copy" button to copy it to your clipboard, then paste it into the ticket.

How to Use the Fixed Comment Tool

This tool is for all other standard ticket comments, such as accessory exchanges, software updates, or closing a ticket for other reasons.

1

Select a Category

In the left-hand menu, click on the category that matches your situation (e.g., "Accessory Exchange").

2

Find and Copy the Comment

The relevant comments will appear on the right. Find the exact comment you need and click its "Copy" button.

Best Practices

Always Use the Tools First

These tools should be your primary way of creating descriptions and comments. This guarantees consistency across all 10 stores.

Report Missing Items

If a part, service, or standard comment you need is not in the tools, please notify your manager. This is valuable feedback that helps us keep the tools up-to-date for everyone.

Avoid Modifying the Text

The goal is to use the exact text copied from the tools. Please do not change the wording unless it's absolutely necessary to add a specific detail (like a unique part number).
